JavaScript está mesmo dominando o mundo?



Dizem pela internet que o JavaScript é a linguagem para a todas governar, mas referências a O senhor dos anéis a parte, será mesmo que essa é a linguagem do futuro? Vocês acham que o JavaScript é a linguagem mais usada no mundo (ou vai ser em breve)? Vale a pena focar somente nela de agora em diante?
